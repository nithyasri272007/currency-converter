# 💱 Currency Converter

I built this currency converter during my semester break to improve my practical coding skills.  
It takes an amount and converts it from one currency to another using real-time data from an API.

## 🚀 Features

- Convert currency in real-time  
- Swap between currencies (⇄)  
- Auto update when input or selection changes  
- Simple and clean user interface  

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- Exchange Rate API  

## 📸 Preview

![Currency Converter](screenshot.png)


## 📌 How to Use

1. Enter the amount  
2. Select the currency you want to convert from  
3. Select the currency you want to convert to  
4. The result will be shown instantly  


## 📂 Files in this project

- `index.html` → structure  
- `style.css` → design  
- `script.js` → logic and API handling
